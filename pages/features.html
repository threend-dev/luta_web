01

로그인 / 회원가입

기능 설명

LUTA 앱에 접속하기 위한 인증 화면입니다. 간편 로그인(구글, 네이버, 카카오톡)과 이메일/비밀번호 방식의 로그인을 지원합니다. 신규 사용자는 회원가입 페이지에서 계정을 생성할 수 있습니다.

사용 방법

  1. 앱을 실행하면 로그인 화면이 표시됩니다.

    간편 로그인(구글, 네이버, 카카오톡) 또는 이메일/비밀번호로 로그인합니다.

  2. 계정이 없는 경우 '회원가입' 버튼을 눌러 새 계정을 생성합니다.
  3. 실사용자 확인을 위하여 입력하신 이메일 인증이 필요합니다.

    인증메일은 스팸 폴더에 있을 수 있으니 확인해주세요.

  4. 이메일 인증이 정상적으로 완료되면 자동으로 로그인됩니다.
02

홈 화면

기능 설명

앱의 메인 허브 화면입니다. 각 주요 기능(팀 관리, 라인업, 경기장, 경기 결과 등)으로 이동할 수 있는 메뉴 버튼들이 배치되어 있습니다.

사용 방법

  1. 로그인 후 자동으로 홈 화면이 표시됩니다.

    원하는 기능의 버튼을 탭하여 해당 화면으로 이동합니다.

  2. 사이드 메뉴에서 각 화면으로 빠르게 이동할 수 있습니다.
03

팀 관리

기능 설명

팀을 생성하고 관리하는 화면입니다. 새로운 팀을 만들거나, 기존 팀에 가입하고, 선수 명단을 관리할 수 있습니다. 각 팀에는 고유한 6자리 팀 코드가 부여되어 다른 멤버를 쉽게 초대할 수 있습니다.

주요 기능

  • 팀 생성 및 팀명 설정
  • 6자리 고유 팀 코드 생성 - 팀 가입을 위해 팀원에게 공유
  • 선수 등록 (이름, 등번호, 포지션)
  • 팀 멤버 초대 및 관리
  • 프리미엄 요금제 구독

사용 방법

  1. 홈 화면에서 '팀 관리' 메뉴를 선택합니다.

    새 팀을 만들려면 팀 명을 입력하고 '팀 생성' 버튼을 탭합니다.

    경기 진행을 위해 라인업을 구성할 팀원들을 추가하고 포지션 및 등번호 등을 관리합니다.

  2. 팀원을 초대하려면 우측상단의 초대 코드를 팀원에게 공유하고, 팀원은 해당 코드를 입력하여 팀에 참여합니다.

    선수목록은 팀 마스터가 경기를 진행하기 위한 선수 정보이며, 팀원목록은 팀에 가입하여 실시간 경기상황 및 경기결과 등을 공유할 수 있는 팀원 목록입니다.

  3. 선수 추가 버튼을 클릭하여 해당 선수의 등번호 및 포지션 등을 입력합니다.
  4. 팀 마스터가 프리미엄 요금제를 구독하면 전체 팀원이 프리미엄 기능을 이용할 수 있습니다.
04

라인업

기능 설명

경기 전 타순과 수비 포지션을 편성하는 라인업 화면입니다. 편성한 라인업은 저장하여 재사용할 수 있고, 팀원 간 실시간으로 동기화됩니다.

주요 기능

  • 타순 편성
  • 수비 포지션 배정
  • 대기 선수 편성
  • 라인업 저장 및 팀원 간 실시간 동기화

사용 방법

  1. 홈 화면에서 '라인업' 메뉴를 선택합니다.
  2. 등록된 선수 목록에서 타순에 배치할 선수를 선택합니다.
  3. 각 선수의 포지션을 지정합니다.

    대기 선수도 편성할 수 있습니다.

    라인업 편성이 완료되면 저장합니다.

05

경기장 (실시간 기록)

기능 설명

실시간으로 야구 경기를 기록하는 화면입니다. 팀 마스터는 경기가 진행중인 경기상황을 경기장 화면에서 직접 타석 처리 결과를 입력할 수 있으며, 타석 결과에 따라 스코어 및 투구 및 타격의 개인 기록이 자동으로 저장되며, 폰과 태블릿 모두에 최적화된 반응형 UI를 제공합니다.

주요 기능

  • 시각적 야구 필드 인터페이스
  • 실시간 점수 업데이트
  • 선수 포지셔닝 표시
  • 경기 단계별 자동 화면 전환
  • 타격 결과를 팀원에게 실시간으로 공유
  • 폰/태블릿 반응형 UI

사용 방법

  1. 홈 화면에서 '경기장' 메뉴를 선택합니다.

    상단의 메뉴에서 경기 시작을 선택하면 홈팀의 선공, 후공을 선택할 수 있습니다.

  2. 타자 마커를 클릭하면 투구 및 타격 결과를 입력할 수 있습니다.

    투수의 투구 결과(볼, 스트라이크)를 선택하면 볼카운트가 반영되고, 타자의 타격이 발생하면 '타격' 버튼을 클릭하여 실제 타구 지점을 지정합니다.

  3. 타구 지점으로 수비수를 선택하면 해당 수비수의 타구 처리 결과를 선택할 수 있으며, 수비수 외의 필드를 선택하면 타자의 진루 상황에 따라 타격 결과를 입력할 수 있습니다.

    파울은 타구 지점 선택 시 실제 파울 영역을 선택 하면 파울 및 파울아웃을 선택할 수 있습니다.

  4. 홈런은 경기장 최상단의 홈런존을 선택하면 홈런상황으로 처리됩니다.

    홈런 및 그 외의 득점 상황이 발생하면 볼카운트 보드위에 '득점처리' 버튼이 생성되며, 사용자가 직접 득점처리 상황을 입력할 수 있습니다.

  5. 타자의 타점은 경기 상황(실책 등)에 따라 타자의 타점으로 인정될 수도, 안될 수도 있으므로 사용자가 직접 판단하여 입력할 수 있습니다.

    타점 및 득점은 일괄처리 또는 주자별로 개별처리가 가능합니다.

  6. 타자 교체는 대기타석 박스위의 '타자 교체' 버튼을 클릭하여 진행할 수 있습니다.

    타자 교체 버튼 클릭 시 현재 대기 중인 선수 목록이 표시되며, 교체할 선수를 선택합니다.

  7. 수비 교체는 경기장 내의 수비수 중 교체할 포지션의 마커를 선택하고, 대기선수 명단에서 선수 교체를 진행하거나 현재 수비 중인 선수와 포지션 교체를 진행할 수 있습니다.

    대기 선수와 교체하거나 포지션을 교체할 경우 교체 가능한 선수 목록이 표시되며, 교체할 선수를 선택합니다.

  8. 타자 및 수비 교체할 선수를 선택하면 최종적으로 교체 대상을 확인 후 교체를 진행합니다.

    타자 및 수비 교체는 홈팀만 가능합니다.

06

경기 결과

기능 설명

지난 경기의 결과를 조회하는 화면입니다. 연도별, 월별 필터를 사용하여 원하는 경기를 찾아볼 수 있으며, 각 경기의 상세 정보(스코어, 이닝별 기록 등)를 확인할 수 있습니다.

주요 기능

  • 경기 이력 목록 조회
  • 연도/월별 필터링
  • 경기 상세 정보 보기

사용 방법

  1. 홈 화면에서 '경기 결과' 메뉴를 선택합니다.

    상단의 필터를 사용하여 원하는 기간을 설정합니다.

    경기 목록에서 원하는 경기를 탭하면 상세 정보를 볼 수 있습니다.

  2. 팀 마스터는 경기결과에 대해서 등판한 투수의 승리/패전/세이브 결과를 기록할 수 있습니다.
07

기록실

기능 설명

현재 진행 중인 경기의 상세 기록을 관리하는 공간입니다. 홈팀에 대한 기록과 원정팀에 대한 기록이 각각 별도의 탭으로 구분되어 있으며, 팀원들은 실시간으로 경기 상황에 따른 기록을 확인할 수 있습니다. 홈팀에 대한 교체 상황만 반영되며, 원정팀에 대한 교체 상황은 반영되지 않습니다.

사용 방법

  1. 홈팀(My Team) 탭에서 홈팀의 타자/투수 기록을 확인합니다.
  2. 원정팀(Away Team) 탭에서 원정팀의 타자/투수 기록을 확인합니다.
08

설정

기능 설명

앱의 전반적인 설정을 관리하는 화면입니다. 앱의 테마 및 그라운드 배경 이미지와 마커, 최대 이닝 수 설정 등의 옵션을 제공합니다. 도움말 페이지에서 앱 사용법에 대한 안내를 확인할 수 있습니다.

주요 기능

  • 앱 테마 설정
  • 경기장 배경 및 홈/어웨이 팀의 마커 설정
  • 최대 이닝 수 설정 (팀 마스터 전용)
  • 도움말 페이지
  • 앱 정보 확인

사용 방법

  1. 홈 화면에서 '설정' 메뉴를 선택합니다.
  2. 야구장 배경 및 타격 위치 마커 설정을 선택합니다.
  3. 앱의 테마 및 홈팀과 원정팀 마커 설정을 선택합니다.
  4. 회원 탈퇴를 원할 경우, '회원 탈퇴' 메뉴를 선택하여 탈퇴 절차를 진행합니다.
